# Audio Player

Web audio player thuần HTML/CSS/JS, chạy tốt trên điện thoại và máy tính.

## File chính
- `index.html`
- `style.css`
- `app.js`

## Tính năng
- Phát file audio cục bộ
- Playlist nhiều bài
- Kéo thả file vào app
- Kéo thả để đổi thứ tự bài hát trong playlist
- Tìm kiếm bài hát trong danh sách
- Tải từng bài nhạc về máy
- Tua bài, chỉnh âm lượng
- Đổi tốc độ phát
- Lặp 1 bài
- Visualizer
- Responsive

## Cách chạy
Chỉ cần mở `index.html` bằng trình duyệt hoặc dùng Live Server trong VS Code.
